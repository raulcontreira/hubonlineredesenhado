Desenvolvimento de Interfaces

Professor Titular: M.sc Wendell Cruzeiro wendell.cruzeiro@ceub.edu.br

Ementa da Disciplina

A disciplina apresenta os fundamentos da Interação Humano-Computador (IHC), com foco na criação de interfaces digitais centradas no usuário. Aborda os conceitos de User Experience (UX) e User Interface (UI), introduzindo princípios de design de interação e usabilidade. Capacita o estudante a desenvolver aplicações web por meio das tecnologias HTML, CSS e JavaScript, além de explorar bibliotecas e frameworks para criar interfaces antigas responsivas, acessíveis e interativas.

Habilidades e Competências

Principais Habilidades a serem Desenvolvidas:

O aluno será capaz de analisar e interpretar os fundamentos de IHC, avaliar criticamente a usabilidade de interfaces e projetar interfaces digitais responsivas e acessíveis. Além disso, aprenderá a codificar páginas web estruturadas com HTML, aplicar regras de estilo com CSS, e criar interatividade com JavaScript. O uso de frameworks como Bootstrap e a colaboração em equipe com Git também são pontos importantes.

Competências Adquiridas:

Ao final, o estudante deverá compreender os fundamentos de IHC, identificar e aplicar princípios de UX e UI, e utilizar corretamente HTML e CSS. Será competente para implementar funcionalidades interativas com JavaScript básico, utilizar frameworks modernos para otimizar o desenvolvimento e trabalhar isoladamente trabalhar em equipe em projetos colaborativos.

Conteúdo Programático

  • Unidade 1: Interação Humano-Computador (IHC): conceitos, histórico, princípios, ergonomia, normas e padrões de usabilidade.
  • Unidade 2: User Experience (UX) e User Interface (UI): fundamentos, heurísticas de Nielsen, design centrado no usuário, arquitetura da informação, acessibilidade e responsividade.
  • Unidade 3: HTML (Hypertext Markup Language): estrutura básica, semântica, tags estruturais e formulários acessíveis.
  • Unidade 4: CSS (Cascading Style Sheets): seletores, propriedades, cores, fontes, layout com Flexbox e Grid, responsividade e especificidade.
  • Unidade 5: Linguagem JavaScript: variáveis, operadores, estruturas de controle, funções, manipulação do DOM, eventos, validações.
  • Unidade 6: Frameworks CSS: introdução e uso de ferramentas como Bootstrap e Tailwind CSS no desenvolvimento de interfaces responsivas.
  • Unidade 7: Principais bibliotecas JavaScript: introdução ao uso de bibliotecas modernas como React, Vue.js e Angular no contexto de desenvolvimento front-end com foco em UX/UI.

Instrumento de Avaliação

A avaliação é contínua e distribuída. As atividades práticas na plataforma online correspondem a 30%, enquanto o projeto final (site funcional e responsivo) vale outros 30%. O restante é dividido entre trabalhos (10%), perfil profissional (LinkedIn/GitHub - 20%) e assiduidade (10%). A frequência mínima obrigatória é de 75% e a menção mínima para aprovação é MM (Médio).
